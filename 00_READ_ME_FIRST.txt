⚡ БЫСТРАЯ УСТАНОВКА (3 МИНУТЫ):

1. Скачайте все файлы в один файл
2. Распакуйте в папку: crypto-miner/
3. Создайте папку templates/ и скопируйте туда HTML файлы
4. Откройте терминал в папке crypto-miner/
5. Выполните команды:

   python3 -m venv venv
   source venv/bin/activate      (Linux/macOS)
   # или
   venv\Scripts\activate         (Windows)
   
   pip install -r requirements.txt
   python app.py

6. Откройте браузер: http://localhost:5000

✅ ГОТОВО! Приложение запущено!

⚠️ ВАЖНЫЕ МОМЕНТЫ:

1. ПАПКА templates/ - Flask ищет HTML файлы ИМ ИМЕННО В ЭТОЙ ПАПКЕ!
   Если её нет или файлы находятся не там - приложение не запустится!

2. Файл app_auth.py ОБЯЗАТЕЛЬНО переименуйте в app.py

3. Первый запуск:
   • Автоматически создается папка data/
   • Создается файл crypto.db (база данных)
   • Создаются все необходимые таблицы
   • Приложение готово к использованию

4. Тестовые учетные данные (создайте сами при регистрации):
   • Username: testuser
   • Email: test@example.com
   • Password: testpass123

📞 ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ:

Ошибка: "TemplateNotFound: register.html"
→ Проверьте, что файлы HTML в папке templates/

Ошибка: "No module named 'flask'"
→ Установите зависимости: pip install -r requirements.txt

Ошибка: "Address already in use"
→ Порт 5000 занят. Измените в app.py на другой (например, 5001)

Ошибка: "database is locked"
→ Удалите файл: rm data/crypto.db (будет пересоздан автоматически)
