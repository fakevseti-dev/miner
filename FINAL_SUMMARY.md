## 🚀 ПОШАГОВАЯ УСТАНОВКА (КОПИРУЙТЕ ПО ПОРЯДКУ!)

### Шаг 1️⃣ - Создание структуры папок

```bash
mkdir crypto-miner
cd crypto-miner

# Создать необходимые папки
mkdir -p templates data logs static/css static/js static/images

# Структура должна быть:
# crypto-miner/
#   ├── templates/          ← В эту папку HTML файлы!
#   ├── data/               ← БД появится здесь
#   ├── logs/
#   └── static/
```

### Шаг 2️⃣ - Скопировать все файлы

```bash
# Скопируйте эти файлы в корень проекта:
cp app_auth.py app.py          # ❗ ВАЖНО: переименуйте!
cp config.py config.py
cp requirements.txt requirements.txt
cp .gitignore .gitignore

# Скопируйте HTML файлы в templates/:
cp register.html templates/register.html
cp login.html templates/login.html
cp miner.html templates/miner.html
```

### Шаг 3️⃣ - Создать виртуальное окружение

```bash
# Linux/macOS
python3 -m venv venv
source venv/bin/activate

# Windows (PowerShell)
python -m venv venv
venv\Scripts\activate
```

### Шаг 4️⃣ - Установить зависимости

```bash
pip install -r requirements.txt
```

### Шаг 5️⃣ - Запустить приложение

```bash
python app.py
```

### Шаг 6️⃣ - Открыть в браузере

```
http://localhost:5000
```

**ВСЕ! Приложение готово! 🎉**

## 🔐 СИСТЕМА АВТОРИЗАЦИИ

### Как работает:

1. **Регистрация** - Создается новый пользователь в БД
2. **Вход** - Проверяется пароль (SHA-256 хеш)
3. **Сессия** - Пользователь получает сессию в Flask
4. **Защита** - `@login_required` проверяет авторизацию


## 🐛 ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ

### Ошибка: "TemplateNotFound"
```bash
# Проверьте, что HTML файлы в templates/
ls templates/
# Должны быть: register.html, login.html, miner.html
```

### Ошибка: "No module named 'flask'"
```bash
# Установите зависимости
pip install -r requirements.txt
```

### Ошибка: "Address already in use"
```bash
# Смените порт в app.py
app.run(debug=DEBUG, port=5001, host='0.0.0.0')  # ← Изменить 5000 на 5001
```

### БД заблокирована
